# Лабораторная работа: Структурные типы данных в C++ (Вариант 18)

Репозиторий содержит решение лабораторной/контрольной работы по теме «Структуры и перечисления в языке C++» для ЮУрГУ. 

## 📋 Задание (Вариант 18)
Определить структурный тип, описывающий компьютерные игры:
* Название игры, студия-разработчик, стоимость подписки, объем аудитории игроков.
* Поддерживаемые платформы (PC, mobile, gamepad).
* Тип/жанр (RPG, стратегия, гонки, авиасимулятор и др.).

### Основные задачи программы:
1.  Инициализировать массив структур внутри программы **20-ю записями**.
2.  Отфильтровать данные: переписать в отдельный массив только **авиасимуляторы с аудиторией от 1 миллиона пользователей**.
3.  Отсортировать новый массив **алгоритмом пузырька** по поддерживаемым платформам.
4.  Реализовать функцию **поиска и вывода подробных данных** по конкретной игре (по названию).
5.  Реализовать функцию **изменения данных игры** (передача по ссылке).
6.  Реализовать функцию вывода **3 самых дорогих игр** (по стоимости подписки).
7.  Поместить в отдельный массив все **стратегии для PC**.
8.  Организовать вывод всех отфильтрованных данных через **единую оберточную функцию** в виде аккуратной таблицы.

---

## 🛠 Требования к реализации (соблюденные в коде)
* **Модульность:** Каждая подзадача вынесена в отдельную изолированную функцию.
* **Перечисления (Enum):** Используются для безопасного хранения типов игр (`GameType`) и платформ (`Platform`).
* **Вложенные структуры:** Реализовано требование по вложенности — структура `GameSpecs` (характеристики жанра и платформы) встроена внутрь основной структуры `Game`.
* **Сортировка:** Классический алгоритм сортировки пузырьком (Bubble Sort).
* **Локализация:** Добавлена поддержка `setlocale` для корректного отображения кириллицы в консоли Windows/Linux.
